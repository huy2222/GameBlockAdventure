# Block Blast Clone
A puzzle game inspired by Block Blast, developed with Unity.  
Players place different block shapes on the board to complete rows or columns, clear lines, and achieve the highest score possible.
## Features
- Drag and drop block placement
- Line clearing mechanics
- Score system
- Combo and bonus effects
- Smooth animations and sound effects
- Responsive UI
- Game over detection
- Simple and addictive gameplay
## Built With
- Unity
- C#
- Unity UI System
## Gameplay
The objective is to place blocks strategically to clear full rows, columns and 3x3.  
The game ends when there is no more space available for the upcoming blocks.
## Architecture & Patterns
This project uses:
- Object-Oriented Programming (OOP)
- Event-driven systems
- Singleton pattern
- Observer/Event pattern
- Modular component-based architecture
